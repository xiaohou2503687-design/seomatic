FAQ

How is this different from Ahrefs/Semrush?

Those are $100+/mo suites for SEO agencies. SEOmatic is a CLI tool for developers. Run it in CI/CD, pre-deploy checks.

Does it work with SPAs (React, Vue)?

Yes. Headless browser rendering analyzes JS-generated content fully.

Can I run it in CI/CD?

npx seomatic audit --ci outputs JSON for GitHub Actions, GitLab CI, or any pipeline.
